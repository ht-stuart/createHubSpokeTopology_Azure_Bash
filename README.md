## 🌐 Azure Hub & Spoke Network Automation (Bash + Azure CLI)

This project is a fully automated Bash script designed to deploy a scalable **Hub & Spoke network topology** in Microsoft Azure using the Azure CLI. It streamlines the creation and configuration of complex networking environments, making enterprise-grade infrastructure deployment faster, consistent, and repeatable.

### 🚀 Key Features

* Automated deployment of **Hub & Spoke architecture**
* Dynamic creation of multiple spoke VNets based on user input
* Built-in validation for Azure subscription and region selection
* Automated provisioning of:

  * Virtual Network Gateway
  * Public IP resources
  * Network Manager & Network Groups
* Centralized connectivity using Azure Network Manager
* Security configuration that **blocks outbound traffic on ports 80 and 443**
* Real-time deployment validation and error handling

### 🧠 Tech Stack

* Bash Scripting
* Azure CLI
* Microsoft Azure Networking (VNets, Gateways, Network Manager)

### 📊 Objective

To simplify and automate the deployment of enterprise cloud networking architectures while enforcing centralized governance and security policies.
